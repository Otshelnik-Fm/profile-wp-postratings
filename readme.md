## Описание:  

Дополнение для WordPress плагина [WP-Recall](https://wordpress.org/plugins/wp-recall/) и плагина [WP-PostRatings](https://wordpress.org/plugins/wp-postratings/) - связывает их проводя тесную интеграцию  

------------------------------
